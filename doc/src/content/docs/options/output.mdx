## Output setting

## Setting Item

- outDir : output directory
- fileName : output file name

## Configuration file

```js
const options = {
  output: {
    outDir: "./mysqlToZod",
    fileName: "schema.ts",
  },
};
module.exports = options;
```

If the setting does not exist, the default setting is applied.

- outDir `./mysqlToZod`
- fileName: `schema.ts`
