# Output - 出力設定

## 設定項目

outputの設定項目は下記2つです。

- outDir : 出力先ディレクトリを設定できます。
- fileName : 出力ファイル名を設定できます。

## 設定ファイル

```js
/** @type {import("./src/options/options").MysqlToZodOption} */
const options = {
  output: {
    outDir: "./mysqlToZod",
    fileName: "schema.ts",
  },
};
module.exports = options;
```

設定が存在しない場合は、デフォルトの設定が適用されます。

- outDir `./mysqlToZod`
- fileName: `schema.ts`
