# tableNames - テーブル名のリスト

## 設定項目

tableNamesの設定項目は、`string[]`型の配列です。

```js
const options = {
  tableNames: ["table1", "table2", "table3"],
};
module.exports = options;
```

ここでテーブル名を指定すると、指定したテーブル名のみをzodSchemaとして出力します。  
指定しない場合、空の配列の場合は、全てのテーブル名をzodSchemaとして出力します。  
デフォルトの設定は全テーブルです。
